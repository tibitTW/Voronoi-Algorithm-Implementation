# Voronoi Diagram Implementation

Voronoi diagram implementation on class "Design and Analysis of Algorithms" at NSYSU, 2023.

## System requirements

1. Install python with version 3.7 or higher

2. Install `tkinter`

    ```shell
    $ pip install tkinter
    ```

    or

    ```shell
    $ pip install -r requirements.txt
    ```

---

## Run

```shell
$ python main.py
```
